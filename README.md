# XTAG to XML
A tool to convert XTAG grammar rules stored in the obsolete .trees format to JSON and XML files. Developed as part of the CRC991 (Project A02) at Heinrich Heine University DÃ¼sseldorf.
For more information, visit http://www.sfb991.uni-duesseldorf.de/en/a02/

## Info
Written in Python 3.4.

How to use:
- python main.py (to use default settings from config.ini)
- python main.py [xtag-directory] [json-directory] [xml-directory] [toggle-single-file]
